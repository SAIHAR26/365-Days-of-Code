# Day 4 - Fibonacci Series

## Problem Statement

Print the first N terms of the Fibonacci Series.

In a Fibonacci sequence, each number is the sum of the previous two numbers.

Example:
0 1 1 2 3 5 8 13 ...

## Languages Used

* C
* Java
* Python

## Concepts Learned

* Loops
* Variables
* Sequence Generation
* Iterative Programming

## Approach

1. Initialize first two numbers as 0 and 1.
2. Print them.
3. Generate the next term by adding the previous two terms.
4. Continue until N terms are printed.

## Time Complexity

O(n)

## Space Complexity

O(1)

## Difficulty

Easy
